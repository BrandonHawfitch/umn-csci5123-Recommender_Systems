# Recommender Systems (CSCI 5123)

### Graduate Academic Course | University of Minnesota
* **Language/Stack:** Python, Jupyter Notebooks
* **Core Focus:** Collaborative Filtering, Evaluation Frameworks, Group Recommendation Algorithms, and Algorithmic Fairness

---

### Course Overview & Scope
This repository contains a collection of advanced assignments and a comprehensive research replication project focused on the design, implementation, and evaluation of modern recommendation systems. The coursework culminated in an extensive group recommendation replication pipeline assessing fairness metrics across varying group dynamics.

### Key Work Completed
* **Algorithmic Replication Study:** Replicated and evaluated group recommendation frameworks, exploring the trade-offs between group utility and individual algorithmic fairness (`replication_submission/`).
* **Evaluation Pipeline:** Built a robust validation and evaluation framework (`Eval_framework.ipynb`) supporting cross-validation folds, metrics generation, and parameter sweeps (Lambda sweeps).
* **Baseline Implementations:** Developed foundational item-based and user-based collaborative filtering models, alongside group aggregation strategies (e.g., Borda count models).

### Skills & Technologies Applied
* **Algorithmic Frameworks:** Collaborative Filtering, Group Recommendation Strategies, Fairness-Aware Recommendation.
* **Evaluation Metrics:** F1-Score at K ($F1@K$), Group Fairness Variance, Group Utility, Borda Count Metrics.
* **Data Engineering & Analysis:** Data matrix restructuring, train/test cross-validation split management (`matrices/`), and JSON-based cohort profiling (`groups/`).
